# Meus ContatosPI

## Status do Projeto

Em desenvolvimento.

## Tecnologias

- Java
- Java Swing
- MySQL
- JDBC
- Git
- GitHub

## Desenvolvedor

Rodrigo Lemos Sá

## Objetivo

Sistema desktop para gerenciamento de contatos, permitindo cadastrar, consultar e organizar informações de contatos de forma simples e eficiente.

## Funcionalidades

- Login do usuário
- Cadastro de contatos
- Consulta de contatos
- Cadastro de categorias
- Integração com banco de dados MySQL
- Interface gráfica em Java Swing
